# FrostpunkGraphicalUpgrade

- Modern borderless window (flip mod, tearing).
- Replaced FXAA with SMAA anti aliasing.
- Accurate FPS limiter.
- Greatly reduced input lag.

## Usage

- Install ReShade 6.6.0+ with full add-on support.
- Copy **FrostpunkGraphicalUpgrade.addon64** and **GraphicalUpgrade** folder in the game folder where **Frostpunk.exe** is (ReShade should be installed in the same folder).

## Compilation

- You can clone [ReShade](https://github.com/crosire/reshade) and add FrostpunkGraphicalUpgrade project to Examples solution, then build the project.
