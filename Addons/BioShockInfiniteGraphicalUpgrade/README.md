# BioshockGraphicalUpgrade

- Modern borderless window (flip mod, tearing).
- Replaced ambient occlusion (AMBIENT OCCLUSION: HIGH and ULTRA) with XeGTAO.
- Replaced FXAA anti aliasing with SMAA.
- Replaced bloom.
- Optionally disable lens flare.
- Optionally disable lens dirt.
- Accurate FPS limiter.
- Upgraded render targets (greatly reduces color banding).
- 10bit color output.
- Greatly reduced input lag.

## Usage

- Install ReShade 6.7.1 or newer with full add-on support.
- Copy **BioShockInfiniteGraphicalUpgrade.addon32** and **GraphicalUpgrade** folder in the game folder where **BioShockInfinite.exe** is (ReShade should be installed in the same folder).

## Compilation

- You can clone [ReShade](https://github.com/crosire/reshade) and add BioShockInfiniteGraphicalUpgrade project to Examples solution, then build the project.
