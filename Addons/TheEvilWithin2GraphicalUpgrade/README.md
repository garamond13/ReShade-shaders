# TheEvilWithin2GraphicalUpgrade

- Modern borderless window (flip mod, tearing).
- Optionally replace TAA with DLSS (DLAA).
- Optionally disable lens dirt.
- Upgraded render targets.
- 10bit color output.
- Greatly reduced input lag.

## Usage

- Install ReShade 6.7.3 or newer with full add-on support.
- Copy **TheEvilWithin2GraphicalUpgrade.addon64**, **nvngx_dlss.dll** and **GraphicalUpgrade** folder into the game folder where **TEW2.exe** is (ReShade should be installed in the same folder).

## Compilation

- You can clone [ReShade](https://github.com/crosire/reshade) and add TheEvilWithin2GraphicalUpgrade project to Examples solution, then build the project.
