# MiddleEarthShadowOfWarGraphicalUpgrade

- Optionally replace TAA with DLSS (DLAA).
- Sharpening filter (Modified AMD FFX CAS).
- Temporal film grain (AMD FFX LFGA).
- Adjustable Tone Response Curve (TRC) (sRGB, gamma).
- Upgraded render targets.
- 10bit color output.
- Greatly reduced input lag.

## Usage

- Install ReShade 6.7.1 or newer with full add-on support.
- Copy **MiddleEarthShadowOfWarGraphicalUpgrade.addon64**, **nvngx_dlss.dll** and **GraphicalUpgrade** folder into the game folder where **ShadowOfWar.exe** is (ReShade should be installed in the same folder).

## Notes

 - "OPTIONS->Display->Scaled Resolution" should be set to "1.00x".

## Compilation

- You can clone [ReShade](https://github.com/crosire/reshade) and add MiddleEarthShadowOfWarGraphicalUpgrade project to Examples solution, then build the project.
