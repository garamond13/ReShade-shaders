# AfterimageGraphicalUpgrade

- Adjustable bloom intensity.
- Sharpening filter (Modified AMD FFX CAS).
- Temporal blue noise film grain (AMD FFX LFGA).
- Adjustable Tone Responce Curve (TRC) (sRGB, gamma).
- Accurate FPS limiter.
- Upgraded render targets.
- Greatly reduced input lag.
- Optionally force v-sync off.

## Usage

- Install ReShade 6.7.3 or newer with full add-on support.
- Copy **AfterimageGraphicalUpgrade.addon64** and **GraphicalUpgrade** folder in the game folder where **Afterimage-Win64-Shipping.exe** is (ReShade should be installed in the same folder).

## Compilation

- You can clone [ReShade](https://github.com/crosire/reshade) and add AfterimageGraphicalUpgrade project to Examples solution, then build the project.
