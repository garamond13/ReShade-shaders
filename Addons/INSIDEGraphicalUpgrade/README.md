# INSIDEGraphicalUpgrade

- Modern borderless window (flip mod, tearing).
- Option to force vsync off.
- 16x anisotropic filtering.
- Sharpening filter (Modified AMD FFX RCAS).
- Adjustable Tone Responce Curve (TRC) (sRGB, gamma).
- Accurate FPS limiter.
- Upgraded render targets.
- 10bit color output.
- Greatly reduced input lag.

## Usage

- Install ReShade 6.7.1 or newer with full add-on support.
- Copy **INSIDEGraphicalUpgrade.addon64** and **GraphicalUpgrade** folder in the game folder where **INSIDE.exe** is (ReShade should be installed in the same folder).

## Compilation

- You can clone [ReShade](https://github.com/crosire/reshade) and add INSIDEGraphicalUpgrade project to Examples solution, then build the project.
